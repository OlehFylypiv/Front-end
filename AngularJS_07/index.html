The uppercase filter format strings to upper case : {{ firstName | uppercase }}

The lowercase filter format strings to lower case : {{ lastName | lowercase }}

The orderBy filter sorts an array :

  • {{ x.name + ', ' + x.country }}
  • {{ x.name + ', ' + x.country }}

The currency filter formats a number as currency :

Price: {{ price | currency }}

The filter selects a subset of an array.
This example displays only the names containing the letter "i" :

  • {{ x }}
This example displays only the names containing the letter "a" :
  • {{ y }}

  • {{ z }}

You can make your own filters by registering a new filter factory function with your module :

  • {{ j | myFormat }}
